# spedicija

Web aplikacija za spediciju i pracenje profita po kontejneru.

## Stack

- Frontend: React + Vite
- Backend: Node.js + Express
- Database: PostgreSQL
- ORM: Prisma

## Lokalno pokretanje

```bash
npm install
npm install --prefix backend
npm install --prefix frontend
npm run dev
```

Backend koristi `backend/.env`, koji nije dio git repozitorijuma.
